# TIL이란?

## 개요
**TIL (Today I Learned)** 는 오늘 내가 배운 것을 기록하는 학습 방식입니다.

## 장점
- 매일 꾸준히 학습하는 습관을 기를 수 있다
- 나중에 다시 찾아볼 수 있는 개인 지식 저장소가 된다
- GitHub 잔디를 채울 수 있다
- 작성하면서 학습 내용을 한 번 더 정리하게 된다

## 작성 팁
1. 완벽하게 작성하려 하지 말고, 일단 기록한다
2. 나만의 언어로 정리한다
3. 코드 예제를 함께 작성하면 좋다
4. 하루에 한 개라도 꾸준히 작성하는 것이 중요하다
