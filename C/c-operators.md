# C언어 연산자

## 개요
C언어의 다양한 연산자 정리. 산술, 논리, 비교, 비트 연산자 등을 다룬다.

## 연산자와 피연산자

- **연산자(Operator)**: 계산을 수행하는 기호 (`+`, `-`, `*`, `/`)
- **피연산자(Operand)**: 연산에 사용되는 값 (5, 2 등)

```c
5 + 2  // 5와 2는 피연산자, +는 연산자
```

## 연산자 우선순위

연산자들 간에는 우선순위가 존재한다. 같은 우선순위면 결합법칙(좌→우)을 따른다.

```c
3 + 5 * 2  // 곱셈 먼저 → 13
3 + 5 + 2  // 좌에서 우로 → 10
```

## 산술 연산자

| 연산자 | 설명 | 예시 |
|--------|------|------|
| `+` | 덧셈 | `3 + 2` → 5 |
| `-` | 뺄셈 | `3 - 2` → 1 |
| `*` | 곱셈 | `3 * 2` → 6 |
| `/` | 나눗셈 (몫) | `7 / 2` → 3 |
| `%` | 나머지 | `7 % 2` → 1 |

### 주의사항

- 정수끼리 나눗셈 → 결과도 정수
- 실수끼리 나눗셈 → 결과도 실수
- `%` 연산자는 정수에만 사용 가능

## 복합 대입 연산자

산술 연산과 대입을 동시에 수행.

```c
int num = 10;
num += 5;   // num = num + 5  → 15
num -= 3;   // num = num - 3  → 12
num *= 2;   // num = num * 2  → 24
num /= 4;   // num = num / 4  → 6
num %= 4;   // num = num % 4  → 2
```

### 피드백 코드

현재 값을 기반으로 새 값을 계산하는 패턴.

```c
int score = 0;
score = score + 10;  // 피드백 코드
score += 10;         // 동일한 동작
```

## 값, 식, 문

- **값(Value)**: 리터럴. ex) `2147483647`, `3.14`, `"Hello"`
- **식(Expression)**: 값으로 귀결되는 것. ex) `3 + 5`, `a * b`
- **문(Statement)**: 컴퓨터가 수행할 명령어

```c
int num = 5;      // 선언문
if (7 == num)     // 조건문, 7 == num은 조건식
```

## 증감 연산자

단항 연산자. 1씩 증가 또는 감소.

| 연산자 | 설명 |
|--------|------|
| `++num` | 전치 증가 (즉시 증가) |
| `num++` | 후치 증가 (다음 줄에 증가) |
| `--num` | 전치 감소 |
| `num--` | 후치 감소 |

```c
int num = 3;
printf("%d", ++num);  // 4 출력 (먼저 증가)
printf("%d", num++);  // 4 출력 (출력 후 증가)
printf("%d", num);    // 5 출력
```

## 논리 연산자

피연산자를 참/거짓으로 평가 후 연산. C에서 0은 거짓, 0 이외는 참.

| 연산자 | 설명 | 팁 |
|--------|------|-----|
| `&&` | 논리 곱 (AND) | 곱셈처럼 생각 |
| `\|\|` | 논리 합 (OR) | 덧셈처럼 생각 |
| `!` | 논리 반전 (NOT) | - |

```c
7 && 0;   // 1 * 0 = 0 (거짓)
-7 || 0;  // 1 + 0 = 1 (참)
!0;       // 1 (참)
```

### Short-Circuit (단락 평가)

- `&&`: 앞이 false면 뒤는 평가하지 않고 false
- `||`: 앞이 true면 뒤는 평가하지 않고 true

## 관계 연산자 (비교 연산자)

| 연산자 | 설명 |
|--------|------|
| `<` | 작다 |
| `>` | 크다 |
| `<=` | 작거나 같다 |
| `>=` | 크거나 같다 |
| `==` | 같다 (등호 2개!) |
| `!=` | 같지 않다 |

```c
// 주의: 연속 비교는 의도대로 동작하지 않음
5 < 3 < 10;  // (5 < 3) < 10 → 0 < 10 → 1 (참)

// 올바른 범위 비교
5 < num && num < 10;
```

## 형변환 연산자

자료형 A에서 B로 변환. 명시적 형변환.

```c
5 / 2;          // 2 (정수 나눗셈)
(float)5 / 2;   // 2.5 (실수 나눗셈)
(int)5.9;       // 5 (소수점 버림)
```

## 삼항 연산자

조건문 if-else의 간단한 대용.

```c
조건 ? 참일_때_값 : 거짓일_때_값
```

```c
int a = 4, b = 5;
int max = (a > b) ? a : b;  // max = 5
```

## sizeof 연산자

자료형 또는 변수의 크기를 바이트 단위로 반환.

```c
sizeof(char);    // 1
sizeof(int);     // 4
sizeof(double);  // 8

int num = 10;
sizeof(num);     // 4
```

## 비트 논리 연산자

피연산자를 2진수로 평가 후 비트별로 연산.

| 연산자 | 설명 |
|--------|------|
| `&` | 비트 AND |
| `\|` | 비트 OR |
| `~` | 비트 NOT (전체 반전) |
| `^` | 비트 XOR (배타적 합) |

```c
int a = 11;  // 1011
int b = 9;   // 1001

a & b;  // 1001 → 9
a | b;  // 1011 → 11
~a;     // ...11110100 → -12
a ^ b;  // 0010 → 2
```

## 비트 이동 연산자

비트를 왼쪽/오른쪽으로 n칸 이동.

| 연산자 | 설명 |
|--------|------|
| `<<` | 왼쪽 시프트 (2^n 곱하기) |
| `>>` | 오른쪽 시프트 (2^n 나누기) |

```c
7 << 2;   // 0111 → 11100 → 28 (7 * 4)
3 << 3;   // 0011 → 11000 → 24 (3 * 8)
8 >> 2;   // 1000 → 0010 → 2  (8 / 4)
```

### 참고

최신 컴파일러는 `* 2`와 `<< 1`의 성능 차이가 거의 없다. 최적화에 너무 집착하지 말자.
